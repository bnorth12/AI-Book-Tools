# Book Editor

An AI-powered book editing and improvement tool that helps refine existing manuscripts through targeted suggestions and collaborative editing.

## Features

- **Project Setup**: Configure your editing project with genre and style preferences
- **Book Input**: Upload and analyze your manuscript text
- **AI Chat Bot**: Interactive AI assistant for editing questions and suggestions
- **Improvements List**: Generate and track specific improvement suggestions
- **Edited Output**: Apply and review suggested changes
- **Request Logging**: Monitor AI usage and track editing history

## Usage

1. Open `BookEditor.html` in your web browser
2. Set up your project in the first tab (API key, genre, etc.)
3. Upload your book manuscript
4. Use the AI chat bot to get editing suggestions
5. Review and apply improvements through the improvements list
6. Export your edited manuscript

## Requirements

- Modern web browser with JavaScript enabled
- xAI API key for AI functionality
- Book manuscript in text format

## File Structure

- `BookEditor.html` - Main editing interface
- `Beyond Prepared.txt` - Sample content or reference material

## Shared Schema

This tool will move toward a shared import/export format using `schema/novel-schema-1.0.json` so it can accept compatible data from `NovelWriter` and `BookDecomposer`.
