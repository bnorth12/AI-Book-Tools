What This Guide Covers

Novel Writer is a browser-based tool for planning, drafting, and refining a full novel with AI assistance while keeping the author in control of every accepted change.

Use this guide as a workflow map if you are starting from scratch, importing a saved session, or checking where a feature lives before running it in the app.

Quick Start

  • Configure your xAI API key and project settings in Tab 1.
  • Build characters and subplots before generating outlines.
  • Generate and refine chapters in sequence to preserve narrative continuity.
  • Export both session JSON and book text regularly while iterating.

Tab 1

API & Story Info

Set up your API key and define basic story details.

How to Use

  • API Key: Enter your xAI API key to enable AI features.
  • Model: Select an AI model such as "grok-4.20-0309-reasoning".
  • Max Tokens: Set the token limit per AI update. The default is 4500.
  • Genre: Choose your novel’s genre.
  • Author Style: Pick an author’s style or fetch more options.
  • Style Guide: View or edit the selected author’s style.
  • Title, Story Arc, General Plot, Setting: Enter manually or click for AI ideas.
  • Number of Chapters: Set how many chapters to plan. The default is 10.
  • Target Chapter Length: Define the target word count per chapter. The default is 2000.

  • Import Session: Upload a JSON file to load a previous session.
