Novel Writer 0.3.3 User Guide

Welcome to *Novel Writer 0.3.3*, a web-based tool for creating and refining novels with AI assistance. This guide explains how to use each tab to build your story from scratch or import an existing session.

Tab 1: API & Story Info

Set up your API key and define basic story details.

How to Use

  • API Key: Enter your xAI API key to enable AI features.
  • Model: Select an AI model (e.g., "grok-4.20-0309-reasoning").
  • Max Tokens: Set the token limit per AI update (default: 4500).
  • Genre: Choose your novel’s genre.
  • Author Style: Pick an author’s style or fetch more options.
  • Style Guide: View or edit the selected author’s style.
  • Title, Story Arc, General Plot, Setting: Enter manually or click for AI ideas.
  • Number of Chapters: Set how many chapters (default: 10).
  • Target Chapter Length: Define word count per chapter (default: 2000).

  • Import Session: Upload a JSON file to load a previous session.
