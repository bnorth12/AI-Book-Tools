## 5. Previously Tracked Issues Resolved

The following bugs were identified and resolved during the sprint (documented in `HerbalBookForge/ISSUES.md`):

| Bug | Root Cause | Resolution |
|---|---|---|
| Only one chapter outline generated | Regex in `parseChaptersFromOutline` too narrow | Regex updated to match all heading styles |
| `useFullOutline is not defined` on Outline Q&A | Variable referenced before definition | Variable defined by reading checkbox state |
| 422 error on Outline Q&A chat | `tools` array included in payload | `tools` array removed from all Outline Q&A payloads |
| Tab switching broken after Outline Q&A fix | Syntax error introduced in `sendSpecificOutlineQuestion` | Syntax error fixed; global functions restored |
| Playwright tests out of sync with unified LLM API | Tests written before unified API refactor | Tests updated to match new payload structure and logic |
