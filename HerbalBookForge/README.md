# Herbal Book Forge

A specialized AI-powered book creation tool designed for crafting herbal medicine, botany, and natural wellness books with structured guidance and expert-level content generation.

## Features

- **Setup**: Configure your herbal book project with API credentials and basic parameters
- **Book Goals**: Define your book's purpose, target audience, and key objectives
- **Outline**: Create comprehensive book structure with chapters and sections
- **Chapters**: Generate detailed chapter content with herbal expertise
- **Drafting**: Refine and expand content with AI assistance
- **Prompts**: Access specialized herbal medicine and botany prompts
- **Safety**: Built-in safety guidelines for herbal content creation
- **Preview**: Review complete book structure before final export

## Usage

1. Open `HerbalBookForge.html` in your web browser
2. Configure your API key and project settings in the Setup tab
3. Define your book goals and target audience
4. Build your outline structure
5. Generate chapter content using specialized herbal prompts
6. Review safety considerations for herbal information
7. Export your completed herbal book project

## Requirements

- Modern web browser with JavaScript enabled
- xAI API key for AI content generation
- Knowledge of herbal medicine or botany (recommended)

## Special Features

- **Herbal Expertise**: Specialized prompts for accurate botanical and herbal information
- **Safety First**: Built-in safety checks and disclaimers for herbal content
- **Structured Approach**: Guided workflow from concept to completion
- **Export/Import**: Save and resume projects with JSON export functionality
